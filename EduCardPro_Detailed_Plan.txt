How to use this document
Each phase is broken into task groups. Every task has three columns: the task name, the full acceptance criteria (what 'done' looks like), and the expected output file or artifact. Paste each task row directly into your AI code-generation tool as the prompt. Work top-to-bottom — tasks within a phase depend on the ones above them.
Stack used throughout:
Frontend: Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui, TanStack Router
Backend: Laravel 11 (PHP 8.3), Sanctum auth, Tenancy for Laravel v3
Database: PostgreSQL 16
Storage: Cloudflare R2 (photos, print files)
Notifications: Meta Cloud API (Messenger), Semaphore (SMS), Resend (email)
PDF / Print: Puppeteer (headless Chrome) for ID cards and report cards
Barcode scanning: QuaggaJS (webcam), USB HID keyboard emulation (plug-and-play)
Hosting: DigitalOcean VPS (Ubuntu 24, Nginx, PHP-FPM)
Error monitoring: Sentry (free tier)
